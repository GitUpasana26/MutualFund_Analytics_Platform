# Mutual Fund NAV Analytics Project

## Objective
To analyze mutual fund NAV trends and compare performance across different schemes using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- APIs (mfapi.in)

## Data Source
NAV data fetched from mutual fund API (mfapi.in)

## Project Workflow
1. Data collection using API
2. Data storage in CSV format
3. Data ingestion and inspection
4. Basic data quality analysis

## Outcome
Built a structured dataset for further financial analysis and visualization.
